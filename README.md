# RFID-Based Smart EV Parking & Charging System

An IoT-enabled smart parking and charging management system featuring ESP32-based hardware simulation, RFID authentication, and a real-time web monitoring dashboard.

---

## 📌 Project Overview
This project delivers an automated solution for Electric Vehicle (EV) parking space management and smart charging station authorization. Using RFID technology, user access is authenticated seamlessly while real-time slot availability, power delivery status, and system analytics are monitored via a dedicated web dashboard.

---

## 🤝 Project Contributions & Breakdown
This project was developed collaboratively with clear division of technical responsibilities:

* **Hardware Architecture & Wokwi Simulation:** 
  * Designed and implemented the core ESP32 firmware and circuit schematics.
  * Simulated full hardware functionality (RFID module, occupancy sensing, slot actuation) using Wokwi.
* **Web Dashboard & Frontend Integration:** 
  * Developed the web dashboard interface for live status tracking, telemetry visualization, and system monitoring.

---

## 🚀 Key Features
* **RFID Access Control:** Secure user verification using RFID tags for charging slot access.
* **Smart Slot Sensing:** Automatic occupancy detection and status updates for EV parking bays.
* **Wokwi Simulated Testing:** Full virtual prototyping of circuit behavior and control logic.
* **IoT Web Dashboard:** Admin control panel for real-time monitoring and analytics.
